Short Circuit Rules for && and ||

If I have an AND, we know not to check for the second condition if the first has already failed. Conversely in OR, if we know the first condition has succeeded we know the operation does as well.

CSEL

This never controls the control to another variable. It's the same thing as a ternary!

CSEL is literally just the ternary syntax : <cond>:<trueput>:<falseput>

While loops

A while loop is just conditional jumps. If we get to the bottom and the condition is true, we jump to the top. If the condition is false, we fall through to the bottom.

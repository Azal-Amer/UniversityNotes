The Multiplication Rule

If operation A can be done in ways and operation B in ways, then sequence (A,B) can be done in ways.

In Plain English: When you have to do several things in sequence, multiply the number of ways to do each thing.

Problem Classification Framework

Step 1: Identify Problem Type

Conditional Probability Problems

Indicators:

  • "Given that..."
  • "If we know..."
  • Sequential events
  • Updated information

Strategy:

  1. Write out P(A|B) = P(A∩B)/P(B)
  2. Draw tree diagram if sequential
  3. Use Bayes' for "reverse" conditionals

Independence Problems

Indicators:

  • Multiple trials
  • "Repeatedly..."
  • No connection between events
  • Product of probabilities

Strategy:

  1. Verify independence by checking P(A∩B) = P(A)P(B)
  2. Use multiplication rule
  3. Consider complement for "at least one" problems

Combinatorial Problems

Indicators:

  • Counting arrangements
  • Selecting items
  • "How many ways..."
  • Ordering/grouping

Strategy:

  1. Ask: Does order matter?
  2. Check: Repetition allowed?
  3. Use appropriate formula:
    • Order matters → Permutation
    • Order doesn't → Combination

Step-By-Step Problem Solving

1. Read & Organize

  • Write given probabilities
  • List conditions
  • Draw diagram if helpful

2. Plan

  • Identify problem type
  • Choose appropriate formula
  • Break into subproblems

3. Execute

  • Follow systematic approach
  • Show clear steps
  • Check intermediate results

4. Verify

  • Answer between 0 and 1?
  • Units correct?
  • Makes intuitive sense?

Common Mistakes & How to Avoid Them

Mistake 1: Wrong Formula Choice

Example: Using when order matters

Prevention:

  • Ask: "Does rearrangement create new outcome?"
  • Try small example first
  • Make organized list

Mistake 2: Independence Assumption

Example: Drawing cards without replacement

Prevention:

  • Check if events affect each other
  • Calculate P(A|B) ≟ P(A)
  • Consider replacement/no replacement

Mistake 3: Probability Computation

Example: Adding instead of multiplying

Prevention:

  • Write complete formula first
  • Show each step
  • Check units and magnitude

Lesson Plan

Lesson Objectives

  • Understand basic for loop syntax and usage
  • Apply for loops to array traversal
  • Implement nested for loops
  • Debug common for loop issues
  • Create programs using for loops with arrays

Timeline (60 minutes)

1. Opening Activity (10 minutes)

  • Send worksheet to students
  • Quick review of arrays from previous homework
  • Students explain one thing they learned about arrays from their video assignment
  • Live coding: Basic array traversal example

2. Core Concept Introduction (15 minutes)

  • For loop syntax breakdown

  • Three key examples:

    // Basic for loop
for (int i = 0; i < 5; i++)

// Array traversal
for (int i = 0; i < array.length; i++)

// Nested loops
for (int i = 0; i < rows; i++)
    for (int j = 0; j < cols; j++)

  • Common pitfalls discussion

  • Live debugging exercise

3. Guided Practice (20 minutes)

  • Basketball Stats Challenge
    • Start with simple array traversal
    • Build to more complex operations
    • Connect to their previous homework
  • Pattern Recognition Exercise
    • Simple pattern first
    • Progress to more complex pattern
  • Real-time feedback and adjustments

4. Independent Practice (10 minutes)

  • Students work on challenge problem
  • Provide support as needed
  • Group discussion of solutions

5. Closing (5 minutes)

  • Review key concepts
  • Preview homework
  • Answer remaining questions

Contingency Plans

  • If students struggle with basic syntax: More examples with array printing
  • If concepts are mastered quickly: Move to challenge problems
  • If engagement drops: Switch to competitive element

Assessment Strategies

  • Verbal checks for understanding
  • Code completion exercises
  • Pattern prediction challenges
  • Real-time debugging participation

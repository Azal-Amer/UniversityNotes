Implicit Cast

Definition (Implicit Cast).

A smaller datatype is "lumped into" the bigger datatype
Ex:

  • Int to a double -> Double
  • Double to a String -> String
  • Anything to a String -> String

public class studying {  
    public static void main(String[] args){  
        int x = 5;  
        double y = 10.5;  
          
        System.out.println(x+y);  
//        Expected output: 15.5 (double)  
        x = (int)(x+y);  
//        PEMDAS, R-L  
//        "the variable x, is equal to: the integer form of (x plus y).  
//        Works because it adds x+y, which is first a double,  
//        before the int cast is used up  
        //        x = (int) x+y;  
//        Works because x is already an int, and adding y turns it back into a double  
//  
        System.out.println(x);  
//        Expected output: 15 (int)  
    }  
}

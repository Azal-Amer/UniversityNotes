Lesson Plan

Objectives

  • Master array traversal with for loops
  • Understand how to work with adjacent elements
  • Build confidence in solving complex array problems
  • Practice breaking down problems into smaller steps

Timeline (60 minutes)

1. Warm-up & Review (10 minutes)

  • Quick review of basic for loops with arrays

    • Write on board: for (int i = 0; i < arr.length; i++)
    • Ask: "What does each part do?"

  • Group Activity: Have students trace this loop:

    int[] nums = {1, 7, 8, 2};
for (int i = nums.length-1; i >=0 ; i—) {
    System.out.print(nums[i] + " ");
}

{"hello","goodbye","potato"}

2. Adjacent Elements Concept (15 minutes)

  • Introduce Part 1 from worksheet
  • Live code printAdjacent() together
    • Draw array visualization on board
    • Show relationship between i and i+1
  • Students complete arraySum() independently
  • Group discussion: Common patterns when working with arrays

3. Building the Solution (20 minutes)

  • Work through Part 2 exercises
    • Demo checkSum() together
    • Students work on firstPairSum10() in pairs
    • Class discusses different approaches
  • Identify pattern: "We're combining array traversal with a condition check"

4. Challenge Implementation (10 minutes)

  • Present final challenge: hasAdjacentSum10()
  • Class brainstorm:
    • What parts from earlier can we use?
    • What edge cases should we consider?
  • Students start implementation

5. Wrap-up (5 minutes)

  • Review key concepts
  • Share successful strategies
  • Preview homework/next steps

Teaching Strategies

Visual Aids

Array:  [3, 7, 8, 2]
Index:   0  1  2  3
Pairs:   3,7
         7,8
         8,2

Key Questions for Students

  • "How do we know when to stop checking pairs?"
  • "What happens if we try to access index i+1 at the end?"
  • "How can we make our code more efficient?"

Common Pitfalls to Address

  • Array bounds errors
  • Off-by-one errors in loops
  • Forgetting to return a value
  • Not considering edge cases

Checkpoints

After each section, verify students can:

  1. Print adjacent elements correctly
  2. Add numbers in an array
  3. Compare adjacent values
  4. Implement the final solution

Extension Activities

For students who finish early:

  • Modify the problem to find three adjacent numbers that sum to 15
  • Find the largest sum of any adjacent pair
  • Implement the solution using a while loop instead

  • To know if acceleration is changing, if your tangent line isn't equal to the curve at that point, then acceleration isn't constant

Converting from Position to Velocity Graph

  1. Label the different sections of graph
  2. Identify if its a curve or a line
    1. If it's a line, the velocity is constant
    2. If it's a curve, velocity is changing
  3. Identify the behavior of the curve's start and end. If we wanna see the slope, draw a tangent line

Converting from Velocity to Position Graph

  1. Label the different sections of graph
  2. Mark where the overall shape has a positive velocity (above x-axis) or negative (below x-axis)
    1. The position only goes down when we have a negative velocity
    2. When there is a positive velocity, the position goes up

Drawing Velocity Graphs from Position Graphs is, frustratingly, more an art than a science. If you can recognize up to the last point, you know as much as needed for the exam
Before doing any problem regarding PVT graphs (Position, velocity, time), DRAW THE PICTURES!!.

I like to visualize Position, velocity, and time graphs as stacked,

Position
Velocity
Acceleration

This visualization is useful because whether we go from position to velocity, or velocity to acceleration, we use the same steps. In fact, if I need to translate a velocity curve to acceleration, I'd do it by drawing as though it was a position curve translated to velocity.

The same principle then applies going up the levels. Going from velocity to position is the same as going from acceleration to velocity. You'd follow our same steps.
